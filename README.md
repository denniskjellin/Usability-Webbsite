# DinResa -  Transportation service - created with HTML, CSS & JavaScript

## Live version
https://webbusabilitybydenniskjellin.netlify.app/

## Project focus
The assigment for this project was to create a website with focus on usability. The idea of the website is to give a service for booking transportation on a daily, weekly or monthly basis - the website have focus on beeing easy to navigate and handle in general, to make it as good as possible for a variety of people with different needs.

### DinResa
Is a fictional company created for this project.
